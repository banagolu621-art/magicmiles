# Magic Miles – AI Restaurant Assistant

An AI-powered restaurant website that helps customers browse the menu, ask questions, reserve tables, and receive intelligent support.

## Live Demo

https://magicmiles.vercel.app

## GitHub

https://github.com/banagolu621-art/magicmiles

## Features

- AI Restaurant Chatbot
- Menu Browsing
- Table Reservations
- FAQs
- Contact Support
- Responsive Design
- n8n Workflow Automation
- Supabase Backend

## Tech Stack

- React
- TypeScript
- Vite
- Lovable
- Supabase
- n8n
- OpenAI API
- Vercel

## Installation

```bash
git clone https://github.com/banagolu621-art/magicmiles.git
cd magicmiles
npm install
npm run dev
```

## Deployment

Hosted on Vercel.
